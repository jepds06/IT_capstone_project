# SupplyEase

A web-based supply chain management system built as a capstone project.

---

## Tech Stack

- **Frontend:** Nuxt 3, Vue 3, Tailwind CSS, Nuxt UI
- **Backend:** PHP Laravel
- **Database:** MySQL

---

## Setup

### Frontend
```bash
cd client
npm install
npm run dev
```

### Backend
```bash
cd server
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```
